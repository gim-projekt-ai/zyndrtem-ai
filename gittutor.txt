szybki tutorial git
-------------------------

git status - sprawdza stan
git pull origin master - ściąga zmiany
git add - dodaje plik do repo
git push -u origin master - dodaje zmiany do chmury
git commit -m "wiadomość" - akceptuje zmiany
git commit -a -m "wiad" - --------------||------------, wszystkie!
